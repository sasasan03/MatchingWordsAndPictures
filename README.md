# MatchingWordsAndPictures<br>
##生徒の自立活動用に作ったサンプルアプリ<br>
言語聴覚士の方からのアドバイスを元に制作<br>
<<どんなアプリか？>><br>
１.左側に人の画像が出てくる
2.右側には左側の人物に該当する名前ボタン
3.画像と名前が一致したボタンを押すと正解の音楽が流れる（自分で探す）。5回正解すると、生徒の好きな音楽が数秒流れる。画像が更新される。
4.間違っていた場合は、不正解の音楽が流れ、画像は更新されない。
